# niamhramadhaan.github.io

Live at **https://niamhramadhaan.github.io**

Plain HTML, CSS, and JavaScript — no frameworks, no build step. Push to
`main` and GitHub Pages serves it.

## Structure

- `index.html` — the whole site (hero, products, experience, skills, contact)
- `css/tokens.css` — design tokens (parchment canvas, ink-blue accent, serif/mono pairing)
- `css/main.css` — layout, components, motion
- `js/main.js` — progressive enhancement only: scroll reveals, nav state, cursor spotlight
- `PRODUCT.md` — design strategy & principles for the site

## Editing

All content lives directly in `index.html`. Colors, type, and spacing are
CSS custom properties in `css/tokens.css`.
